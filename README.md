# dworkspace

Standard structure for Dev directories!


### How to run

```bash
1. git clone https://github.com/bmalbusca/dworkspace.git
2. bash ./workspaces.sh

```

### The result
```bash

Workspaces
├── documents
│   └── publish
├── <external-project>
└── internal
    ├── lab
    │   ├── foo
    │   └── <language-framework>
    ├── notes
    └── projects
        ├── current
        ├── late-2019
        ├── late-2020
        ├── late-2021
        └── late-2022



```
